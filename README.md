# Task 6: Create and Evaluate Strong Passwords

## Objective
To understand password strength and evaluate different passwords using an online tool.

## Tool Used
passwordmeter.com

## Passwords Tested
1. password123 (Weak)
2. Nisha123 (Medium)
3. Ni$ha@2026!Secure (Strong)

## Observations
- Weak passwords are short and predictable
- Medium passwords include numbers but still guessable
- Strong passwords include symbols, numbers, uppercase, lowercase, and length

## Best Practices
- Use at least 12 characters
- Mix uppercase and lowercase letters
- Include numbers and symbols
- Avoid names and dictionary words
- Use unique passwords

## Outcome
Learned how password complexity improves security and protects against brute force attacks.
